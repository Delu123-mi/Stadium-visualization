# \# OpenGL Stadium Project

# 

# This project is a 3D stadium model developed using C++ and OpenGL (FreeGLUT) as part of a Computer Graphics.

# 

# \# Features

# \- Oval football pitch

# \- Athletics running track

# \- Tiered seating bowl

# \- VIP seating under the roof

# \- Safety railing and stone facade

# \- Stadium name text (ASTU STADIUM)

# \- Team benches and entrance gates

# 

# \# Technologies Used

# \- C++

# \- OpenGL

# \- FreeGLUT

# \- GLU

# 

# \# How to Run

# 1\. Install FreeGLUT

# 2\. Compile using a C++ compiler (e.g., MinGW)

# 3\. Run the executable
